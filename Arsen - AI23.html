KONFIDENSINTERVALL

Fr att berkna konfidensintervallet fr medelvrdet anvnds fljande formeln:

${\bar x} \pm Z_{\alpha/2} \frac{}{\sqrt{n}}$

In[]:
mean = Versicolor["petal_length"].mean() # Mdelvrde fr stickprovet (x bar)
std = Versicolor["petal_length"].std() # Standardavvikelse fr stickprovet ()
n = len(Versicolor["petal_length"]) # Antal fr stickprovet (n)
alpha = 0.05 # Signifikansniv
t_crit = t.ppf(1-alpha/2, n-1) # Kritiskt t-vrde
sem = std / np.sqrt(n) # Medelvrdesstandardfel 
upper = mean + t_crit * sem # vre grns
lower = mean - t_crit * sem # Undre grns
# Avrundning till 3 decimal
lower = np.round(lower, 3)
upper = np.round(upper, 3)
mean = np.round(mean, 3)
print(lower, " < ", mean, " < ", upper)

4.126  <  4.26  <  4.394

Fr att visualisera frdelningen av petal_length i Versicolor-stickprovet och markera konfidensintervallet, skapades fljande graf:

In[]:
# Skapa en graf med Seaborn
plt.figure(figsize=(8, 6))
sns.histplot(Versicolor["petal_length"], kde=True, color='skyblue', bins=20)
plt.axvline(mean, color='red', linestyle='dashed', linewidth=2, label='Medelvrde')
plt.axvline(upper, color='green', linestyle='dashed', linewidth=2, label='Konfidensintervall')
plt.axvline(lower, color='green', linestyle='dashed', linewidth=2)
# Markera medelvrdet och konfidensintervallens grnser
plt.text(mean, plt.ylim()[1] * 0.9, f'Medelvrde: {mean:.2f}', color='red', ha='center')
plt.text(upper, plt.ylim()[1] * 0.8, f'{alpha*100}% CI', color='green', ha='center')
plt.text(lower, plt.ylim()[1] * 0.8, f'{alpha*100}% CI', color='green', ha='center')
plt.legend()
plt.show()

Detta konfidensintervall p 95% ger oss ett intervall dr vi med rimlig skerhet frvntar oss att det sanna medelvrdet fr petal_length i populationen finns.

  • Det 95% konfidensintervallet fr medelvrdet strcker sig frn 4.126 till 4.394.
  • Vi r 95% skra p att det verkliga medelvrdet ligger inom detta intervall baserat p det observerade stickprovsmedelvrdet, standardavvikelsen och stickprovets storlek.
